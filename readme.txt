Add a simple list of shortcodes to WordPress in order to display the current user information.

== Description ==
This plugin display the Current user information as well as its meta information.

This plugin is very flexible you can get users meta by meta key, it will display the value that user meta property.

This shortcodes works only for  loged in users.

This plugin has the following shortcodes

<code>[displayusers_meta meta="meta key"]</code> - Display the loggedin users metas value according to parameter<br>
<code>[display_firstname]</code> - Display the Loggedin user firstname<br>
<code>[display_lastname]</code> - Display the Loggedin user lastname<br>
<code>[display_displayname]</code> - Display the Loggedin user name<br>
<code>[display_userid]</code> - Display the Loggedin userid
<code>[display_username]</code> - Display the user name ID<br>
<code>[display_useremail]</code> - Display the Loggedin useremail<br>




== Installation ==
1. Upload the plugin to "wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.

== Frequently Asked Questions ==
= How to use this plugin? = 
use code in your page Like this <code>[display_firstname]</code>

= How to get meta values? = 
Pass the meta key name parameter to the shortcode Like this <code>[displayusers_meta meta="meta key"]</code>
